Exploring WordPress Customization

Pamela Avila

July 6, 2026

For this project, I decided to use the Astra theme and Wordfence Security plugin. I picked Astra because its simple, lightweight codebase offers a great basis for performance and also makes changes later on easier without adding unnecessary visual elements. I chose Wordfence as it is very important to have strong access control, endpoint firewall implementation, and login security for those features protections are required for any web environment. The theme totally altered the visual layout, typography, and frontend structure of the site, whereas the plugin added a security dashboard to the backend for monitoring traffic and blocking malicious activity.
